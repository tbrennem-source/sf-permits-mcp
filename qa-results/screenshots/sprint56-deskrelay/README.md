# Sprint 56 DeskRelay Screenshots

Screenshots were captured via Claude in Chrome MCP extension during visual QA.
Due to intermittent extension issues ("Detached" / "Cannot access chrome-extension://"),
not all screenshots could be saved to disk. Visual verification was done in real-time
in the browser with page text extraction as fallback.

## What was visually verified:

### Staging
- Landing page: title, project card, violation card, search bar
- Analyze preview: fork comparison (OTC vs In-House), locked cards
- Violation context: "Violation Lookup Mode" banner
- Beta request form: fields, honeypot, submission confirmation
- Login with referral_source parameter

### Prod
- Landing page: title, header nav, search bar, recent chips, action chips
- Search results: "75 robin hood dr" - 21 permits, property intel panel
- Property report: permit detail, severity score, plan review routing, related permits, planning records
- Login redirect for logged-in user
