Electronic Plan Review (EPR) Requirements

All plans must be submitted electronically as of January 1, 2024

  • Vector-based PDF required: CAD-produced vector lines, not scanned images. Non-compliant = rejection.
  • TrueType/OpenType fonts: Searchable text required. SHX fonts need OCR post-processing.
  • Single consolidated PDF: One PDF per document type. No individual sheet files.
  • Full 1:1 scale: Print at 100% scale from authoring software.
  • Minimum 22" x 34" sheets: ANSI D or Arch D minimum. Signage permits: 11x17. Supplements: 8.5x11.
  • File size < 250MB: Per upload. Site permit addenda: 350MB limit.
  • PDF unlocked: No encryption, passwords, or security restrictions.
  • No digital certificate signatures: Use scanned graphic signatures. DocuSign/Adobe Sign certificates lock the PDF.
  • Back Check page: DBI standardized Back Check page appended to plan set.

Pre-Submission Checklist

EPR Resubmittal Process

When corrections are required during plan review:

  • EPR-023: Download marked-up plans from Bluebeam Studio session
    ⚠️ Common mistake: Only reviewing comments from one reviewer — multiple agencies comment simultaneously
  • EPR-024: Address ALL plan check comments
    ⚠️ Common mistake: Missing comments from agencies that review later in the cycle (e.g., DPH, SFFD)
  • EPR-025: Add revision clouds on ALL changed items
    ⚠️ Common mistake: Submitting corrected sheets without revision clouds — reviewer cannot find what changed
  • EPR-026: Upload only changed sheets for corrections
    ⚠️ Common mistake: Uploading the full plan set for a correction response — slows review and confuses version tracking

DA-02 Common Deficiency Areas

  • Site Arrival: Insufficient accessible parking (1 per 25 spaces first 100, then 1 per 50)
  • Exterior Route: Excessive cross-slope on exterior walkways (max 1:48)
  • Entrance/Exit: Door hardware requiring tight grasping or twisting — must be lever or push/pull type
  • Interior Route: Protruding wall-mounted objects (fire extinguishers, signage) not detected
  • Vertical Access: Elevator does not meet CBC 11B-407 requirements for call buttons and cab size
  • Restrooms: Toilet centerline not 16-18 inches from side wall — most common restroom correction
  • Special Occupancy Features: Service counter too high — must provide 36" max height section at least 36" wide
  • Signage: Tactile room signs not at correct height (48-60 inches above floor) or missing braille

Mitigation Strategies

  • Use the completeness checklist (tier1/completeness-checklist.json) before submission
  • Include a Back Check page in all plan sets

  • Verify plan description matches permit application exactly
  • Consider CASp (Certified Access Specialist) inspection — reduces ADA correction rate from ~38% to ~10%
  • Submit DA-02 checklist with initial application (most common ADA correction is missing DA-02)
